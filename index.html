Paper Autocomplete Demo

Examples

Local Data-Source Binding

Directly bind a local array to paper-autocomplete. The component internally will filter suggestions based on the component attribute settings.

```html <paper-autocomplete label="Select State" id="input-local" no-label-float="true"></paper-autocomplete> ``` ```js HTMLImports.whenReady(function(){ var inputLocal=document.querySelector('#input-local'); inputLocal.source=theLocalStatesArray; }); ```
LOCAL DATA BINDING
The component internally remaps the passed array to the template binding array (which will be an array of objects with a `text` and `value` properties). The passed array can be a simple array of strings.

Public properties, `textProperty` and `valueProperty`, can be declaratively set in the element attributes, allowing to bind an arbitrary object array to the component's source property.
```html <paper-autocomplete text-property="userName" value-property="id"></paper-autocomplete> ```

Remote Data-Source Binding

Remote bind to the github user api. Remote data binding delegates the responsibility of filtering the data source to the calling app. Set a listener for the autocomplete.change event and pass the suggestions back to the component. The change event is fired based on the min-length attribute setting.

```html <paper-autocomplete label="Select User" id="input-remote" remote-source="true" min-length="2"></paper-autocomplete> ``` ```js function onChange(event){ var input=document.querySelector('#input-remote'); var search=event.detail.option.text; var url='https://api.github.com/search/users?q=' + search + '+in:login'; var req= new XMLHttpRequest(); req.open('GET',encodeURI(url)); req.onload = function() { if (req.status === 200) { var data=JSON.parse(req.response); var arr=data.items.map(function(obj){ return {text:obj.login,value:obj.login}; }); input.suggestions(arr); } }; req.send(); } ```
REMOTE DATA BINDING
The filtered suggestions is expected to be mapped to an array of text/value pairs before being passed to the component.

Selected Suggestion

Setup a selected event listener to get the value of a suggested selection.

```js document.addEventListener('autocomplete-selected',onSelect); function onSelect(event){ var selected=event.detail.text; } ```
SELECTED EVENT
Returns {id,value,text,target,option} for the selection.

 

Public API

Properties

label {String} The text input placeholder default: ""
noLabelFloat {Boolean} Set to true to disable the floating label default: false
source {Array} Local array data source default: []
remoteSource {Boolean} Binds to a remote data source default: false
textProperty {String} property of local datasource to as the text property default: "text"
valueProperty {String} property of local datasource to as the value property default: "value"
value {String} The current/selected value of the input default: ""
text {String} The current/selected text of the input default: ""
disableShowClear {Boolean} Disable showing the clear X button default: false
minLength {Number} Minimum length to trigger suggestions default: 1
required {Boolean} Set to true to mark the input as required default: false
autoValidate {Boolean} Set to true to auto-validate the input value default: false
errorMessage {String} The error message to display when the input is invalid default: ""
eventNamespace {String} Event type separator default: "-"

Methods

getValue {String} Gets the current value of the input params: none
getOption {Object} Gets the current text/value option of the input params: none
setOption {void} Sets the current text/value option of the input params: {Object}
disable {void} disables the input params: none
enable {void} enables the input params: none
validate {void} validates the input params: none
suggestions {void} sets the component's current suggestions params: {Array}
clear {void} clears the current input params: none
reset {void} resets the current input params: none

Events

autocomplete.selected {id,value,text,target,option} Fires when a selection is made
autocomplete.change {id,value,text,target,option} Fires on input change
autocomplete.focus {id,value,text,target,option} Fires on input focus
autocomplete.blur {id,value,text,target,option} Fires on input blur
autocomplete.reset {id,value,text,target,option} Fires on input reset/clear
